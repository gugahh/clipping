# Clipping
Aplicativo de clipping, que consome matérias de *NewsAPI.org*.

Realiza a tradução automaticamente dos artigos do inglês para o português do Brasil.
Os assuntos são configuráveis (editando *clipping.py*). Utiliza o argostranslate para traduzir.

Para executar, apenas clique em **clipping.bat** (no Windows).

*Criado por Gustavo Santos: gugahh.br@gmail.com*

## Instalação
- Garanta que você tem o Python 3 instalado em seu computador: [Download Python](https://www.python.org/downloads/)
- Execute, no diretório da aplicação:

>   **pip install -r requirements.txt**

- Execute a seguir:

>   **python instala_tradutor.py**

- Verifique se algum navegador está configurado para ler arquivos Markdown ((.md)), como este aqui. Se não estiver, instale a extensão *MD Reader*.
- Tudo está pronto. Agora é só configurar os seus assuntos (vide linha 65 de clipping.py).
- Clique em **clipping.bat**.
